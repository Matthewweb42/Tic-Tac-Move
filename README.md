# Tic-Tac-Move

A twist on classic Tic-Tac-Toe where players can move their pieces after placing them!

## Game Jam Jan 2026

This game was created for the Game Jam Jan 2026 challenge.

## How to Play

1. **Placement Phase**: 
   - Players take turns placing their pieces (X or O) on the board
   - Each player places exactly 3 pieces on the board
   - Try to get three in a row during placement to win immediately!

2. **Movement Phase**:
   - After all 6 pieces are placed, the game enters move mode
   - Players can now move their pieces to adjacent empty cells
   - Adjacent includes horizontal, vertical, and diagonal neighbors
   - Select your piece, then click an adjacent empty cell to move
   - First player to get three in a row wins!

## How to Run

Simply open `index.html` in a web browser. No installation or build process required!

## Features

- Clean, modern UI with gradient background
- Responsive design that works on mobile and desktop
- Visual feedback for selected pieces and available moves
- Automatic mode switching after placement phase
- Reset button to start a new game

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## Game Rules

- The board is a standard 3x3 grid
- Each player gets 3 pieces
- Win condition: Get three of your pieces in a row (horizontal, vertical, or diagonal)
- Pieces can only be moved to adjacent empty cells
- Diagonal moves are allowed

Enjoy playing Tic-Tac-Move!
